# auto-open-preview README

Run a predefined command when open some type of files. 

By default it is .typ -> typst-preview.preview.

## Extension Settings

Use `auto-open-preview.commands` to set a `lang_id -> command` map. Both key and values are strings.

Default value: 

```json
{
  "typst": "typst-preview.preview"
}
```
